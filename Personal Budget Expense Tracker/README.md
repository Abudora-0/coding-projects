# FinTrack - Personal Budget Expense Tracker

A modern, responsive web application for tracking personal income and expenses with visual analytics, budget management, and detailed reporting features.

## 🎯 Features

- **Dashboard Overview** - Quick stats including total income, expenses, balance, and savings rate
- **Expense Tracking** - Log transactions with categories, amounts, and dates
- **Visual Analytics** - Interactive pie charts and line graphs for expense breakdown and monthly trends
- **Budget Management** - Set spending limits for different categories and monitor progress
- **Transaction Management** - Search, filter, and export transactions
- **Reports** - Detailed financial reports and spending analytics
- **Dark/Light Mode** - Toggle between dark and light themes for comfortable viewing
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Local Storage** - All data saved locally in your browser (no cloud uploads)

## 💾 Data Storage

All your financial data is stored locally in your browser's localStorage. This means:
- ✅ Your data never leaves your computer
- ✅ No account creation needed
- ✅ Instant access without waiting for servers
- ⚠️ Data is cleared if you clear browser cache/cookies
- ⚠️ Data is unique to each browser profile

### Backup Your Data
To backup your data, use the export feature in the Reports section to download your transactions as a file.

## 🐛 Troubleshooting

### Data Not Saving?
- Check if your browser allows localStorage
- Try a different browser
- Clear browser cache and try again

### Charts Not Displaying?
- Ensure JavaScript is enabled
- Check browser console for errors (F12)
- Clear cache and refresh the page

### Mobile Layout Issues?
- Try rotating your device
- Zoom out slightly for better view
- Use the latest browser version
